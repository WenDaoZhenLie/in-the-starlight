## 定位
主题： 追星里的情绪安放 + 自我照顾 + 清醒喜欢
人设： 
	- 同担 / 理性粉 / 温柔陪伴型
	- 不营销，不卖惨，不站队
	- 温柔、清冷、高级、克制
视觉体系： 统一背景 + 极简字体 + 大留白（爱心只用于结尾引导页）

## 目标
用“温柔但清醒”的内容持续吸引追星女孩，并在信任建立后变现：
	- 数字产品（记录本/壁纸/文案包）
    - 文案定制服务（回复、置顶、朋友圈、小作文）
	- 轻咨询（追星节奏/情绪内耗梳理）

近期增长目标（30天内）
	- 粉丝：391 → 2000
	- 稳定日更或隔日更
	- 连续再跑出 1 条 1w+（爆款延伸）

## 内容
追星女孩的温柔安全区：不拉踩、不戾气、不证明，也不否定喜欢。
起号期：
  1. 入场宣言-价值观沉淀，情绪共鸣（收藏/关注）
  2. 互动扩散-幸福瞬间（评论/推荐）
  3. 自我照顾-方法论（深化人设）
稳定期：
从评论提炼主题

### “追星偏贬义”的应对策略（表达体系）
关键原则：不为“追星”辩护，而是升级表达

1. 从“追星”改成更正向、更可被理解的词：
	•	“喜欢一个人”
	•	“热爱与陪伴”
	•	“情绪支撑”
	•	“精神寄托”
	•	“青春的一束光”
	•	“把喜欢放回生活”

2. 避免踩雷的表达
	•	“追星让我活下去 / 没他们不行”（评论区可接住，但不要正文强调）
	•	“粉圈大战”“撕逼”细节（容易引战）
	•	过度饭圈用语（粉籍/控评/打投）大段讲
	•	“必须/应该/正确追星方式”（像说教）

3. 推荐的价值观锚点
	•	喜欢不是考勤
	•	不拿自己证明爱
	•	追星可以热烈，但生活要稳
	•	先照顾身体与现生
	•	“喜欢之后，我依然是我”
